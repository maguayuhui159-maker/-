# 匠脉锡雕三创赛平台 - 角色权限与功能需求文档

## 1. 核心角色定义

为了满足“匠脉”平台的业务闭环，系统将用户体系划分为以下三个核心角色（Role）：

1. **管理员 (Admin)**：平台的运营方与最高管理者。
2. **课程创建者 (Uploader/Teacher)**：锡雕技艺传承人、文化传播者或专业讲师。
3. **普通学生 (Student)**：对非遗锡雕感兴趣的年轻一代、学徒或普通消费者。

---

## 2. 角色功能差异与权限矩阵

### 2.1 管理员 (Admin)
**核心诉求**：全局把控平台内容质量，统筹用户运转，监控平台营收与流量数据。

*   **全站数据看板**：查看平台总用户数、总课程数、订单交易流水、AI 服务调用频率等宏观数据。
*   **用户生命周期管理**：
    *   查看所有用户的列表（学生、课程创建者）。
    *   封禁违规用户、重置用户密码。
    *   审批普通用户升级为“课程创建者”的入驻申请。
*   **内容审核机制**：
    *   **课程审核**：审核课程创建者提交的新课程，决定是否允许上架（包含视频合规性查验）。
    *   **作品审核**：审核学生上传到“广场”或申请售卖的实物作品。
*   **资金与平台订单管理**：处理退款申请，查看所有课程与作品的购买订单记录。
*   **系统与 AI 参数配置**：配置首页轮播图、公告，调整 AI 模型接口所需的 Secret Key 等基础设置。

### 2.2 课程创建者 (Uploader / Teacher)
**核心诉求**：高效发布教学内容，实现非遗文化变现，管理自己的学员资产。

*   **专属创作中心**：
    *   **课程管理**：使用所见即所得的编辑器创建课程，上传视频、文本和图片简介。
    *   设定课程价格（支持免费试看和付费解锁）。
    *   上下架自己的课程，查看某节课的审核状态（草稿、审核中、已发布、被驳回）。
*   **学员与互动管理**：
    *   查看购买了自己课程的“学生”列表。
    *   回复学生在课程下方的提问和评论。
*   **收益与提现**：查看个人的课程卖出份数与营收总额，申请资金提现到绑定的银行卡或支付宝。
*   **作品发布特权**：作为专业人士，也可以将自己的精致锡雕作品发布到交易广场进行展示或标价售卖。

### 2.3 普通学生 (Student)
**核心诉求**：学习非遗知识，利用 AI 辅助设计，展示学习成果，购买周边。

*   **学习中心 (C端核心)**：
    *   浏览、搜索和购买平台上已上架的课程。
    *   记录课程播放进度，收藏喜欢的课程。
    *   在课程下方发表评价及提问。
*   **AI 智能设计与文创**：
    *   使用 AI 工具生成锡雕纹样。
    *   购买平台官方或者其他用户售卖的锡雕周边/工具包。
*   **个人作品成果转化**：
    *   **上传作品**：完成学习后，将自己的手工成品拍照上传。
    *   **申请上架售卖**：如果对作品满意，可以勾选“申请售卖”并标价，提交给管理员审核；审核通过后作品将展示在全站的公共集市（Store）供他人购买。
*   **个人中心**：管理个人收货地址、查看购买订单列表，及个人的学习时长成就。

## 3. 业务流转示例（角色交互）

以**“课程发布与学习”**流程为例：
1. **[上传者]** 登录后台 -> 创建《锡雕入门：打錾技法》课程 -> 标价 99 元 -> 提交审核。
2. **[管理员]** 登录管理控制台 -> 在“课程审核”列表中看到新课程 -> 观看后点击【通过】。
3. **[学生]** 浏览前台商城 -> 看到该课程 -> 支付 99 元 -> 开始学习 -> 学完后制作了一个锡雕小碗。
4. **[学生]** 拍照上传小碗 -> 点击【申请售卖】，标价 50 元。
5. **[管理员]** 审核该作品照片，确认不违规 -> 审核通过。
6. 其他用户即可在广场购买该学生的锡雕小碗。
